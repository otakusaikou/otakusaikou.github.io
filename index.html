Point Cloud Demo

Here comes some demonstrations of the result from my master thesis, using potree - a web based point cloud viewer

A brief metholodogy can be found from this link: 2017_ISRS_Poster.

NTU Admin raw RGBI

Point cloud of NTU Admin building. Captured by ground LiDAR scanner Z+F IMAGER 5010. Data contains RGB and intensity information.

NTU Admin RGB

Result of my master thesis. This data was a conbination of LiDAR and photogrammetry techniques. By integration of LiDAR 3D coordinates and high resolution photos contain rich optical informaiton, we significantly improved the spatial and color quality of the 3D point cloud.

NTU CERB RGB

Point cloud of NTU CERB. Data generated using optical photos taken with drone flying around the building. 3D point cloud generated using resulting aerial photos and commertial software - Pix4D.

NTU CERB IR

IR painted point cloud. Processed using XYZ-RGB point cloud and IR photos with camera exposure calculated by space resection approach.
